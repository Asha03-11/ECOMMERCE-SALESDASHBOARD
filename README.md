 📊 E-commerce Sales Dashboard

Project Overview
This project is a Power BI dashboard that analyzes e-commerce sales data. It provides meaningful insights into business performance and helps in decision-making.

Objectives
- Analyze total sales and profit
- Identify top-selling products
- Understand customer behavior
- Track regional performance
- Visualize trends over time

 Dataset Information
The dataset includes:
- Order details
- Customer information
- Product categories
- Sales and profit data
- Region-wise performance

 Features of Dashboard
- Interactive slicers and filters
- KPI indicators (Revenue, Profit, Orders)
- Sales trend analysis
- Category-wise performance
- Region-wise analysis

🛠️ Tools Used
- Power BI
- Data Visualization Techniques
- Data Cleaning & Transformation
